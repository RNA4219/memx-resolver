# Code of Conduct

本プロジェクトは、参加者全員に対して安全で建設的なコラボレーション環境を提供することを目的とします。

## 期待される行動

- 相手の背景・経験・価値観を尊重する
- 技術的根拠に基づいて議論する
- フィードバックは具体的かつ敬意を持って行う
- 攻撃的・差別的・威圧的な言動をしない

## 禁止行為

- ハラスメント、差別、侮辱、脅迫
- 個人情報の無断公開
- プロジェクト運営を妨げる継続的な迷惑行為

## 問い合わせ先

- 行動規範に関する問い合わせ・報告: `conduct@memx.local`
- セキュリティ報告は `SECURITY.md` の窓口を利用してください。
