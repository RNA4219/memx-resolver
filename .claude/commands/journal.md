# Add Journal Entry

Add a time-series log entry to memx journal store.

## Usage

```
/journal --scope <scope> <log content>
```

## Required Arguments

- `--scope`: Working scope (e.g., `project:memx`, `session:abc`, `self`)

## What it does

1. Creates a journal entry with the specified working scope
2. Journal entries are time-ordered logs (progress, decisions, events)
3. Can be filtered by scope later

## Example

```
/journal --scope project:memx Completed CRUD implementation for journal store
```

## Implementation

Use the Bash tool to run:

```bash
cd ../../memx-core/memx_spec_v3/go && go run ./cmd/mem in journal --title "<auto-generated title>" --body "<log content>" --scope "<scope>" --sensitivity internal
```
