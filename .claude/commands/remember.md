# Remember Information

Store information in memx short-term memory for later retrieval.

## Usage

```
/remember <information to store>
```

## What it does

1. Saves the provided information to memx's short-term memory store
2. Returns a note ID for future reference
3. Information becomes searchable via `/recall`

## Example

```
/remember The API endpoint for user authentication is /v1/auth/login
```

## Implementation

Use the Bash tool to run:

```bash
go run ./cmd/mem in short --title "<auto-generated title>" --body "<user provided information>" --sensitivity internal
```

Generate a concise title from the content if not obvious.
