# Add Knowledge

Add a knowledge base entry to memx knowledge store.

## Usage

```
/knowledge --scope <scope> <knowledge content>
```

## Required Arguments

- `--scope`: Working scope (e.g., `glossary`, `patterns`, `architecture`)

## Optional Arguments

- `--pinned`: Pin the knowledge for quick access

## What it does

1. Creates a knowledge entry with the specified working scope
2. Knowledge entries are time-independent facts, definitions, policies
3. Can be pinned for quick reference

## Example

```
/knowledge --scope glossary typed_ref format: memx:{type}:{provider}:{id}
```

```
/knowledge --scope architecture --pinned memx uses 4 SQLite stores: short, journal, knowledge, archive
```

## Implementation

Use the Bash tool to run:

```bash
cd ../../memx-core/memx_spec_v3/go && go run ./cmd/mem in knowledge --title "<auto-generated title>" --body "<knowledge content>" --scope "<scope>" [--pinned] --sensitivity internal
```
